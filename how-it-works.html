The War Room, layer by layer

Each block is a real part of the system — agents at the top do the work, the three surfaces in the middle are where you operate, and the bottom layer is the infrastructure. The gold line on the right marks the one place sensitive data is allowed to live.

The three layers

Built so nothing critical has a single point of failure.

Layer 1 · Agents

The team that does the work

Cipher engineers the prompts, Momentum produces and distributes content, Conversion runs leads and follow-up, and Crypto/Architect handle Web3 and systems. They're operating prompts with guardrails — education-first, no guarantees, defensive security only.

Layer 2 · Surfaces

Where you actually work

The Prompt Lab (build & test), the Content Studio (media & campaigns), and the Owner's Console (run the business, see results). These are ordinary, fast web apps — the "war room" is the experience, not the plumbing.

Layer 3 · Infrastructure

Decentralized where it can be, controlled where it must be

Inference, storage, and agent memory run on the decentralized stack (Chutes, Hippius, Ditto) for resilience and cost. The one exception: regulated personal data lives on controlled, encrypted, deletable infrastructure — because the law requires it. That single boundary is what keeps the whole system both free and compliant.
